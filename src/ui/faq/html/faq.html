FAQ

Why does this add-on needs permissions?

Access your data for www.youtube.com
- needed for removing users/channels you have blocked on YouTube™, adding the 'x'-button, etc. (rather obvious)

Access browser tabs
- needed for keeping track of the YouTube™-Tabs and the Add-on setting page tab.

Store unlimited amount of client-side data
- needed for blocking an unlimited number of users/channels.

I found a bug, have an idea for a new feature or a different question, what now?

Contact us under yt-cleaner@protonmail.com. If you found a bug a picture is always helpful.

Do you collect any data?

We DON'T collect data from you in any form!

The addon dosen't work.

Check if you got the newest version of Channel Blocker. If you are using a phone, we are sorry but this add-on doesn't work on mobile browsers.
If you got the newest version of our add-on and it still doesn't work please use our bug report feature by visiting a YouTube™ page you experience problems on. Then press right-click with your mouse and choose the "Report a bug on this page" option. Afterwards fill out the empty fields in the newly created bug report tab, save the bug report and send it to yt-cleaner@protonmail.com.
